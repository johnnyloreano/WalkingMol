Navegador de proteína

Para começar, você deve selecionar a opção ABRIR BUSCADOR DE PROTEÍNA, onde você será redirecionado para uma página onde poderá escolher uma proteína. Para escolher a proteína, você deve digitar o CÓDIGO IDENTIFICADOR de tal.Esse código é formado por 4 caracteres, sendo o primeiro um número e, os outros, letras Um exemplo de código é o 1ZDD. Caso ocorra um erro, será informado ao usuário, e ele deverá tentar novamente. É importante notar que o usuário deverá ter acesso a internet, caso seja a primeira vez que busque determinada proteína. Se ela já foi buscada, será salva no seu computador. Uma vez concluída a pesquisa, o usuário será direcionado a uma página onde conterá o título, com o código da proteína buscada, e, logo abaixo, a estrutura da proteína. Para melhor compreensão, usar fones de ouvidos. Para iniciar a navegação, o usuário precisará clicar no botão "Abrir proteína", e o leitor de tela deverá dizer "Visualizador de proteína. Use o tab para começar!". A partir desse momento, o usuário poderá usar o TAB para se movimentar na proteína.

Dentro da proteína

Uma vez que o usuário esteja dentro da proteína, você poderá usar o TAB para movimentar para o próximo aminoácido, e SHIFT + TAB para voltar um aminoácido para trás. Utilize o ESPAÇO para ouvir mais informações sobre o aminoácido. Caso esteja dentro de uma hélice ou fita, o usuário será informado disso.
