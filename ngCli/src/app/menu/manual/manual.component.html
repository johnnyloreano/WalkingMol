Navegador de proteína

Escolher a proteína

Para escolher a proteína, você deve selecionar a opção ABRIR BUSCADOR DE PROTEÍNA no menu principal, onde você será redirecionado para uma página para realizar essa tarefa. Será necessário digitar o CÓDIGO IDENTIFICADOR da proteína. Um exemplo de código é o 1ZDD. Caso ocorra um erro, será informado ao usuário, e ele deverá tentar novamente. Caso seja a primeira vez que busque determinada proteína, é importante lembrar que o usuário deverá ter acesso a internet. Se ela já foi buscada, será salva no seu computador, e não será necessário estar conectado à internet.

Menu do navegador

Uma vez concluída a pesquisa de proteína, o usuário terá um menu por onde deverá escolher a próxima ação. No menu estão as seguintes tarefas: ENTRAR NA PROTEÍNA, por onde poderá começar a navegação, e VOLTAR AO MENU ANTERIOR, para voltar para o menu principal

Dentro da proteína

Uma vez que o usuário esteja dentro da proteína, você poderá usar o TAB para movimentar para o próximo aminoácido, e SHIFT + TAB para voltar um aminoácido para trás. Utilize o CTRL para ouvir mais informações sobre o aminoácido. Caso esteja dentro de uma hélice ou fita, o usuário será informado disso.
