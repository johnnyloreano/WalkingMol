Bem-vindos à Ferramenta Educativa LABIO! Essa ferramenta foi construída para que você possa compreender a estrutura de moléculas de proteínas simples.

Voltar para título INTRODUÇÃO

No menu principal, você poderá acessar o navegador de proteínas e também o manual.

Voltar para título Menu Principal

Para começar, você deve selecionar a opção ABRIR BUSCADOR DE PROTEÍNA, onde você será redirecionado para uma página onde poderá escolher uma proteína. Para escolher a proteína, você deve digitar o CÓDIGO IDENTIFICADOR de tal. Um exemplo de código é o 1ZDD. Caso ocorra um erro, será informado ao usuário, e ele deverá tentar novamente. É importante notar que o usuário deverá ter acesso a internet, caso seja a primeira vez que busque determinada proteína. Se ela já foi buscada, será salva no seu computador. Uma vez concluída a pesquisa, o usuário será direcionado a uma página onde conterá o título, com o código da proteína buscada, e, logo abaixo, a estrutura da proteína. Será necessário que o usuário desligue, caso possua, o seu leitor de tela, pois a ferramenta já possui um leitor próprio, e use fones de ouvido, para melhor compreensão. Para iniciar a navegação, o usuário precisará usar a tecla TAB, e o leitor de tela deverá dizer ""
